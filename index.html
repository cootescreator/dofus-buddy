⚙ Connexion Google Sheets

Suis ces 5 étapes une seule fois. Tes données seront ensuite sauvegardées dans ton Google Sheet, accessibles depuis n'importe quel appareil.

1
Ouvre Google Sheets, crée un nouveau classeur, nomme-le "Dofus Comptoir".
2
Dans le menu en haut : Extensions → Apps Script. Un éditeur de code s'ouvre.
3
Supprime tout le code existant, puis colle le contenu du fichier apps-script.js fourni avec cette app. Clique sur l'icône 💾 Enregistrer.
4
Clique sur Déployer → Nouveau déploiement. Dans "Type" choisis Application Web. Dans "Accès" choisis Tout le monde. Clique Déployer, autorise les permissions Google.
5
Copie l'URL de déploiement (elle commence par https://script.google.com/macros/s/…/exec) et colle-la ici :
